## 🔒 چرا این پروژه ساخته شد؟

✅ بدون نیاز به نصب VPN  
✅ همیشه متصل به آخرین دامنه سالم  
✅ قابل نصب به عنوان اپلیکیشن (PWA)  
✅ اجرا فقط با یک کلیک از صفحه خانه گوشی یا دسکتاپ
